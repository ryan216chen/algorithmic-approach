# 線性篩的實作原理

(題目來源:https://zerojudge.tw/ShowProblem?problemid=d117)
原理:整數唯一分解定理

任何大於1的自然數，其可以被質數組合而成，且質數的類型和數量不變。
相對於埃式篩，線性篩避免了重複指派非質數的操作。

主要是判斷一個數i是否為質數，然後跟已經判斷過為質數的vector一一產生x去標記非質數。

大致可以寫成 x = p * i; p代表判斷過為質數的數，從vector取的

假設x = 2 * 2 * 3，可以看成p = 2，i = 2 * 3，x = 12。
然後在i % p == 0的時候就直接跳出換判斷下一個i是否為質數。

## 為甚麼可以直接跳出而保證不會漏掉某個非質數沒標記到?

x = 2 * 2 * 3，因為這時p = 2在i = 2 * 3裡面，可以看成一個順序2 * 2 * 3像升冪排列，
如果這時候不break而是繼續嘗試下一個質數，在這裡是3也就是x = 3 * 2 * 3，x = 18，
但如果按照升冪排列來看，18會在哪裡生成?

x = 2 * 3 * 3，也就是18會在判斷i = 9的時候測試的p是2會生成，那我這時候2 * 2 * 3不跳出就會先標記一次下一輪的數18為false，
而到判斷i = 9的時候又會再標記一次，因此若i % p == 0要跳出去嘗試下一個i。

可以按照升冪排列來看，如果一個數在程式中不是由升冪相乘組成的，那會造成重複計算到同一數字。
根據整數唯一定理，x可以由很多質數相乘組成，因為可能會沒標記到的是下一輪的p2形成的x，但這時i裡面已經有上一輪的p1，且i又會試到全部數字，
p1 < p2，故如果p2作為最前面最小的升冪排列，會跟i中的p1打架，而p2這組會在後面更大數i2有包含p2的時候，配到p1為p，
總之因為整數唯一定理且i足夠大，i同時也可以被分解成質數相乘，故照升冪排列來看絕對不會漏掉非質數沒標記為false。
